Содержимое пункта Администрирование

Состоит из регистрации администратора: после регистрации свободного входа в программу не будет, доступ будет осуществляться только после корректного ввода логина и пароля. Регистрация: Введите логин, введите пароль и нажмите на кнопку: Регистрация. Под появившейся надписью 'регистрация завершена данные сохранены' нажмите выход.

Содержимое пункта Работа с файлами

Включает в себя два функционала(нажмите на данный текст левой клавишей мыши): Считать текст с фото: Подразумевает под собой чтение текста из любого .jpg;.jpeg;.png;.gif;.bmp файла.(только русский текст) Выбрав курсором мышки данную строку нажмите на левую кнопку мышки. В появившемся окне выберите необходимый вам файл. Нажмите на кнопку открыть. Появиться блокнот с извлечённым текстом.


Прочитать PDF:(Работает только при наличии подключения к сети Internet) Считывает текст с файла PDF и преобразует его в Аудиокнигу. Выберите данную строку, нажмите на левую кнопку мыши, выберите необходимый файл PDF, можете продолжить работу в приложении. Когда появиться сообщение Аудио книга (название) создана, нажмите ок. Зайдите на Диске C, папка под именем Audio_books в которую сохранился *.mp3 файл (*-название pdf файла), который по необходимости можно будет прослушать. Если случится обрыв с интернетом , появится ошибка об Подключении, нажмите на кнопку ок и повторите попытку.

Содержимое пункта Кнопка добавить

Для создания учётных записей в журнале(базе данных) нажмите на кнопку Добавить вверху слева: появиться окно добавления Организации и Номеров автомобилей либо Ф.И.О посетителей. Введите название организации и Ф.И.О., номер для сохранения нажмите кнопку добавить. Появится сообщени о сохранении организации/в организацию данных. Нажмите ок. После окончания внесенния необходимых данных нажмите кнопку Закрыть. Если Организация ранее была созданна то в неё внесуться данные Номеров автомобилей либо Ф.И.О посетителей. После сохранения данных в главном окне приложения появяться записи с Id а также название организации. Двойно шелчёк на названии Организации позволит просмотреть записи в этой Организации

Содержимое пункта Кнопка Обновить

Что бы вернуться обратно в Главное окно необходимо нажать на кнопку Обновить, правее от кнопки Добавить

Содержимое пункта Кнопка Поиск

Кнопка поиск необходима для поиска организаций либо автотранспорта/посетителей по ранее созданном журнале(базе данных) После нажатия на кнопку Поиск, появится окно поиска. Введите информацию которую вы хотите найти и нажмите на необходимый параметр поиска( кнопки: Поиск по организациям, Поиск авто/Ф.И.О). После нажатия вы попадёте в окно поиска, где отобразиться необходимое(заданое вами ранее) исковая информация. При поиске организации покажет id организации а так-же название организации которое вы искали.Если поисковой организации не будет в журнале(базе данных), под полями ID и организации по поиску (введённые данные) будет пусто, нажмите обновить для возврата в главное меню. Поиск по авто/Ф.И.О осушествляется тем же способом. При нахожденни данных в журнале(базе данных) появится сообщение о том, что данные найдены в (Название организации) ,нажмите ок, отобразиться дата внесения в журнал а так-же организация в которой данная запись(авто/Ф.И.О.) находиться. При отсутствии поисковых данных(авто/Ф.И.О) появится сообщение что таких данных в базе нет. Нажмите ок.

Содержимое пункта Кнопка автопоиск

Кнопка автопоис служит для автоматического определения номера с камеры наблюдения и поиска данного номера в журнале(базе данных). После нажатия отобразиться область автопоиска(в окне Figure 1), для продолжения закройте окно(Figure 1). После закрытия появиться сообщение (авто распознанно/ошибка распознования). При появлении ошибки появится окно для ручного ввода, нажмите кнопку ок, проведите поиск в ручном режиме как было указано выше. Если авто было распознанно появится сообщение(Авто распознанно), В скобках после занка % и до знака % будет номер распозноного авто. Нажмите ок. И так-же как было указано выше при отсутствии данного номера под строками Дата внесения и (распознанный номер) в Организации, будет пусто. Нажмите Обновить, что бы вернуться в главное окно.

Содержимое пункта Кнопка Голос поиск

Осушествляет поиск по журналу(базе данных) номеров авто и посетителей голосовым вводом. После фразы Listenning начинается запись голоса пользователя. После произношения через несколько секунд молчания поступит франа Start recognision, что будет означать что голосовая команда начала обработку. Если распознавание прошло успешно и данные есть в журнале(базе данных) в окне появяться записи как при ручном Поиске. При ошибке распознавания появится сообщение(предложение) ввести данные в ручную. Если в журнале (базе данных) поисковой информации не будет то под полями "Дата внесения" и "... в организации" будет пусто.

Содержимое пункта Кнопка Удалить данные

Служить для удаления данных с организации. Выберите организацию из которой хотите удалить какие либо данные. Зайдите во внутрь это организации двойным шелчком левой клавиши мыши. Выберите необходимые данные которые хотите удалить(одним шелчком мыши нажмите на необходимую строку, она станет подсвечиваться синим) и нажмите на кнопку Удалить данные.Появиться сообщение Удаленны данные и те данные которые вы удалили. Нажмите ок Данные в данной организации будут удалены и Вы вернетесь в главное окно.

Содержимое пункта Кнопка Удалить орг-ю

Служит для удаления организации из журнала(базы данных). По принципу как удалять данные наведите курсор(либо найдите по Поиску) необходимую организацию которую хотите удалить, нажмите на неё одним щелчком левой клавиши мыши, строка будет станет подсвечиваться синим. И нажмите на кнопку Удалить орг-ю. Организация будет удалена и вы овтаматически окажетесь в главном окне.

Содержимое пункта Кнопка История событий

Позволяет искать событие которые осушествлялись в главном окне. Первая колонка показывает действия совершаемые в главном окне. Вторая отображает Ф.И.О либо номер автомобиля, третья показывает дату и время осуществляемого дейстивия. Так же как и в главном окне, можно осуществлять поиск по действиям: Удалено, Добавлено, Ошибка, Поиск. По номерам авто(в формате XXXX-X/Ф.И.О.(в формате Иванов И.И.) Дате(число-01Месяц-Август(пробел)год-2023), если ввести только месяц, то вывведет весь результат за месяц. Если год, то за весь год. Если необходимо за конкретную дату то, между числом и месяцем нет пробела. Указывайте месяц пез падежа.
