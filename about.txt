Меню программы: 
Администрирование, Работа с файлами и Помощь(Под иконкой и надписью 'Сторонние организации') в вверхнем левом углу.

Администрирование: 
Состоит из регистртации администратора: после регистрации свободного входа в прогграмму не будет,
доступ будет осушествляться только после корректного ввода логина и пароля.
Регистрация: Введите логин, Введите пароль и нажмите на кнопку: Регистрация. Под появившейся надписи  'регистрация завершена данные сохранены'
нажмите выход.

Работа с файлами:
Включает в себя два функционала(нажмите на данный текст левой клавишей мыши):
Считать текст с фото:
Подразумевает под собой чтение текста из любого .jpg;.jpeg;.png;.gif;.bmp файла.(только русский текст)
Выбрав курсором мышки данную строку нажмите на левую кнопку мышки.
В появившемся окне выберите необходимый вам файл. Нажмите на кнопку открыть.Появиться блокнот с извлечённым текстом.

Прочитать PDF:(Работает только при наличии подключения к сети Inthernet)
Считавает текст с файла PDF и преобразует его в Аудиокнигу.
Выберите данную строку, нажмите на левую кнопку мыши, выберите необходимый файл PDF, подождите.
Выскачит окно с кнопкой и начнётся голосовая озвучка вашего текста.
Так-же создастся на Диске C папка, под именем Audio_books в которую сохранится *.mp3 файл (*-название pdf файла).
Который по необходимости можно будет прослушать.
Нажатие на кнопку 'Прекратить прослушивание' закончит звуковое чтение и закроет выскачившае окно.


Помощь (Имеет две функции):
1. Справка:
Ознакомление пользователя работы с программой
2. О программе:
Ознакамливает пользователя с разработчиком данного программого обеспечения. 


Основное меню.
Кнопка добавить:
Для создания учётных записей в журнале(базе данных) нажмите на кнопку Добавить вверху слева:
появиться окно добавления Организации и Номеров автомобилей либо Ф.И.О посетителей.
для сохранения нажмите кнопку добавить. Появится сообщени о сохранении организации/в организацию данных. Нажмите ок. 
После окончания внесенния необходимых данных нажмите кнопку Закрыть.
Если Организация ранее была созданна то в неё внесуться данные Номеров автомобилей либо Ф.И.О посетителей.

После сохранения данных в главном окне приложения появяться записи с Id а также название организации.
Двойно шелчёк на названии Организации позволит просмотреть записи в этой Организации

Кнопка Обновить:
Что бы вернуться обратно в Главное окно необходимо нажать на кнопку Обновить, правее от кнопки Добавить

Кнопка Поиск:
Кнопка поиск необходима для поиска организаций либо автотранспорта/посетителей по ранее созданном журнале(базе данных)
После нажатия на кнопку Поиск, появится окно поиска. Введите информацию которую вы хотите найти и нажмите на необходимый
параметр поиска( кнопки: Поиск по организациям, Поиск авто/Ф.И.О). После нажатия вы попадёте в окно поиска, где отобразиться
необходимое(заданое вами ранее) исковая информация. При поиске организации покажет id организации а так-же название организации
которое вы искали.Если поисковой организации не будет в журнале(базе данных), под полями ID и организации по поиску 
(введённые данные) будет пусто, нажмите обновить для возврата в главное меню. Поиск по авто/Ф.И.О осушествляется тем же способом. 
При нахожденни данных в журнале(базе данных) появится сообщение о том, что данные найдены в (Название организации) ,нажмите ок, 
отобразиться дата внесения в журнал а так-же организация в которой данная запись(авто/Ф.И.О.) находиться. 
При отсутствии поисковых данных(авто/Ф.И.О) появится сообщение что таких данных в базе нет. Нажмите ок. 

Кнопка автопоиск:
Кнопка автопоис служит для автоматического определения номера с камеры наблюдения и поиска данного номера в журнале(базе данных).
После нажатия отобразиться область автопоиска(в окне Figure 1), для продолжения закройте окно(Figure 1). После закрытия появиться
сообщение (авто распознанно/ошибка распознования). При появлении ошибки появится окно для ручного ввода,
нажмите кнопку ок, проведите поиск в ручном режиме как было указано выше. Если авто было распознанно появится сообщение(Авто распознанно),
В скобках после занка % и до знака % будет номер распозноного авто. Нажмите ок. И так-же как было указано выше при отсутствии данного
номера под строками Дата внесения и (распознанный номер) в Организации, будет пусто. Нажмите Обновить, что бы вернуться в главное окно.

Кнопка Голос поиск:
Осушествляет поиск по журналу(базе данных) номеров авто и посетителей голосовым вводом. После фразы Listenning начинается запись
голоса пользователя. После произношения через несколько секунд молчания поступит франа Start recognision, что будет означать
что голосовая команда начала обработку. Если распозвование прошло успешно и данные есть в журнале(базе данных) в окне появяться
записи как при ручном Поиске. При ошибке распознования появится сообщение(предложение) ввести данные в ручную. Если в журнале
(базе данных) поисковой информации не будет то под полями "Дата внесения" и "... в организации" будет пусто.

Кнопка Удалить данные:
Служить для удаления данных с организации. Выберите организацию из которой хотите удалить какие либо данные. Зайдите во 
внутрь это организации двойным шелчком левой клавиши мыши. Выберите необходимые данные которые хотите удалить(одним шелчком мыши
нажмите на необходимую строку, она станет подсвечиваться синим) и нажмите на кнопку Удалить данные.Появиться сообщение
Удаленны данные и те данные которые вы удалили. Нажмите ок Данные в данной организации будут удалены и Вы вернетесь
в главное окно.

Кнопка Удалить орг-ю:
Служит для удаления организации из журнала(базы данных). По принципу как удалять данные наведите курсор(либо найдите по Поиску)
необходимую организацию которую хотите удалить, нажмите на неё одним щелчком левой клавиши мыши, строка будет станет подсвечиваться
синим. И нажмите на кнопку Удалить орг-ю. Организация будет удалена и вы овтаматически окажетесь в главном окне. 

Кнопка История событий:
Позволяет искать событие которые осушествлялись в главном окне.  
Первая колонка показывает действия совершаемые в главном окне. Вторая отображает Ф.И.О либо номер автомобиля, третья показывает дату и время 
осуществляемого дейстивия. Так же как и в главном окне, можно осуществлять поиск по действиям, номерам авто/Ф.И.О, и дате.

Окно История событий
Нажмите на иконку 'Поиск в истории'. Появится окно поиска. 
Поск по дате:
Введите число и месяц(с большой буквы, в именительном падеже) слитно.Нажмите поиск по дате. останутся только данные по дате которую вы задали.
Введите название месяца в именительном падеже с большой буквы, нажмите на кнопку , поиск по дате. 
Останутся только данные за тот месяц который вы задали.
Введите год в числовом формате. Нажмите на кнопку, поиск по дате. Останутся только данные за тот год который вы задали. 
 
Поиск по действиям:
Введите действие Искали, нажмите кнопку поиск по действиям. Останутся только данные которые искали по базе данных из основного окна.
Введите действие Добавлено, нажмите кнопку поиск по действиям. Останутся только данные которые Добавили в базу данных из основного окна.
Введите действие Удалены, нажмите кнопку поиск по действиям. Останутся только данные которые удалили из организации из базу данных из основного окна.
Введите действие Удалена, нажмите кнопку поиск по действиям. Останутся только организации которые удалили из базу данных из основного окна.
